# SELFnews 백엔드 API

- 자신이 발행한 포스트를 정기적으로 이메일로 받아볼 수 있는 웹 애플리케이션 백엔드 API 입니다.
- 자신이 공부한 자료를 간단히 요약, 발행, 구독을 통해 이메일 알람을 주기적으로 받아볼 수 있는 서비스입니다.
- 구독한 포스트를 아침, 점심, 저녁 주기적으로 알람을 받아볼 수 있어 메타인지와 학습 개선에 도움을 줍니다.
